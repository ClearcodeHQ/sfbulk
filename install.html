Installation¶

Install with pip¶

If you are using pip, you can install the sfbulk package using the following commands:

pip install git+git://github.com/clearcode/sfbulk#egg=sfbulk

Install from source¶

Download or clone the source from Github and run setup.py install:

git clone https://github.com/clearcode/sfbulk.git
cd sfbulk
python setup.py install

Requirements¶

sfbulk has one external dependency:

requests==2.3.0
