Operation processÂ¶

  1. Initialize bulk.

    from sfbulk import Bulk

bulk = Bulk()

  2. Authorize bulk.

    bulk.login('username',
           'pasword',
           'security_token')

  3. Create job.

    bulk.job_create(u'query', u'Contact')

  4. Create batch.

    data = 'SELECT Id FROM Contact'
bulk.batch_create(data)

  5. Wait untill job finishes.

    while (not bulk.job_is_completed()):
    sleep(10)

  6. Show batch status.

    status = bulk.batch_status()

  7. Show batch result.

    result = bulk.batch_result()

  8. Close job.

    bulk.job_close()
